# Information_resourses 信息获取资源

## 声明

本人为高中生，这是我创建的第一个项目。由于不太熟悉 GitHub，如有不当之处还请指出。

## 简介

由于互联网时代的信息越来越多，找到有用的、高质量的信息源变得越来越难。很多人并不知道有哪些渠道获取信息。

本项目旨在列举那些高质量的信息获取的网站、博主等资源，打破信息壁垒。

本人知道的也有限，欢迎大家集思广益！
